# 📦 INSANE-MEGAPACK

A browser-based collection of **terminal simulators, UI tools, and JavaScript experiments** for learning front-end development.

---

## ⚠️ Disclaimer

Educational project only. No real hacking, system access, or security bypassing.

---

## 🚀 Features

* 💻 Fake Linux terminal system
* 📁 Virtual file system (mkdir, touch, rm)
* 📦 Package manager UI (APT/YUM style)
* 🧠 Nano-style text editor
* 🎮 Right-click inspect / UI overlay tools
* 🔖 Useful bookmarklets

---

## 🛠️ Tech

HTML • CSS • JavaScript

---

## 🎯 Goal

Learn DOM manipulation, UI systems, and browser-based app design.
